# Waonder Cloud Plugin - Available Skills

Welcome to the Waonder Cloud Plugin for Claude Code. This plugin extends your development workflow with Waonder-specific capabilities.

## Available Skills

| Skill | Description |
|-------|-------------|
| `/waonder-cloud:help` | Show this help message |
| `/waonder-cloud:spec` | Create project specifications and milestone documentation |

## Plugin Components

This plugin includes:
- **Skills**: Custom slash commands for common Waonder development tasks
- **Agents**: Specialized AI agents for code review and development
- **Hooks**: Event handlers for automated workflows

## Getting Started

1. Explore the Waonder backend codebase
2. Use skills to automate common tasks
3. Customize by adding your own skills in `skills/`
